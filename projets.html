Application Web de Gestion des Magasins

Cette application web permet la gestion centralisée des stocks, produits et utilisateurs dans un environnement multi-magasins. L'interface intuitive facilite les opérations quotidiennes et l'analyse des données commerciales.

Technologies utilisées:
Oracle APEX Oracle Database SQL PL/SQL HTML/CSS JavaScript

Système de Recommandation de Livres En cours

Développement d'une application web utilisant des algorithmes de machine learning pour analyser les préférences de lecture et recommander des livres personnalisés aux utilisateurs. Le système s'améliore progressivement grâce aux feedbacks utilisateurs.

Technologies utilisées:
React Flask Machine Learning Deep Learning MongoDB Node.js

Site web de Sudoku

Plateforme web interactive permettant aux utilisateurs de générer des grilles de Sudoku personnalisées, de jouer en ligne et d'utiliser un solveur automatique. L'application inclut différents niveaux de difficulté et un système de suivi des performances.

Technologies utilisées:
Java Jakarta EE MySQL Python HTML/CSS JavaScript Servlet/JSP

Smart Cane (canne intelligente)

Conception d'une canne intelligente équipée de capteurs et d'une caméra pour détecter les obstacles et assister les personnes malvoyantes dans leurs déplacements quotidiens. Le système fournit des retours haptiques et audio pour guider l'utilisateur.

Technologies utilisées:
Computer Vision IoT Arduino Raspberry Pi Python OpenCV

Assistant médical intelligent En cours

Développement d'un assistant virtuel utilisant l'intelligence artificielle pour comprendre les symptômes décrits par les utilisateurs et proposer des pistes médicales pertinentes. Le système n'est pas conçu pour remplacer les professionnels de santé mais pour faciliter les premiers échanges.

Technologies utilisées:
Intelligence Artificielle NLP Web Interface Python React TensorFlow

Dashboard pour un CHU

Conception et mise en place de trois tableaux de bord pour différents services du centre hospitalier universitaire de Oujda. Ces tableaux de bord ont permis d'améliorer considérablement la prise de décision médicale.

Technologies utilisées:
Power BI SQL Excel DAX DataViz

Technologies utilisées:
Python TensorFlow NLTK Transformers BERT Hugging Face
